## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Create Greeting File** | 3 greeting styles (Standard, Enthusiastic, Formal) |
| 2 | **Append to File** | Add content to existing files |
| 3 | **Read File** | Display any file contents |
| 4 | **List All Files** | Show all files/folders with emoji icons |
| 5 | **Delete File** | Preview content before deletion with confirmation |
| 6 | **Show My Info** | Create personal profile saved to file |
| 7 | **Exit** | Gracefully exit the program |

### Bonus Features
- 🎨 Colorful emoji-based interface
- ✅ Input validation and error handling
- 📝 File preview before deletion
- 🕐 Timestamp on saved profiles
- 📂 File type icons (.txt 📝, .rs 🦀)
